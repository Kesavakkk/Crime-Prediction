🔴 Real-Time Crime Prediction API

WebSocket-based real-time communication for live crime data updates and alerts

Getting Started

Connect to the WebSocket server using Socket.IO client library:

const socket = io('http://localhost:5000');

Installation

npm install socket.io-client # or <script src="https://cdn.socket.io/4.5.4/socket.io.min.js"></script>

WebSocket Events

LISTEN connect

Triggered when client successfully connects to the server

socket.on('connect', () => { console.log('Connected to server'); });
LISTEN disconnect

Triggered when client disconnects from the server

socket.on('disconnect', () => { console.log('Disconnected from server'); });
LISTEN user_count

Receives the current number of active users

count (integer): Number of active users
socket.on('user_count', (data) => { console.log('Active users:', data.count); });
EMIT request_live_data

Request latest crime statistics from the server

socket.emit('request_live_data');
LISTEN live_data_update

Receives live crime data updates

states (array): List of state names
crimes (array): Crime counts for each state
year (integer): Data year
timestamp (string): ISO timestamp
socket.on('live_data_update', (data) => { console.log('States:', data.states); console.log('Crimes:', data.crimes); console.log('Year:', data.year); console.log('Timestamp:', data.timestamp); });
EMIT crime_alert

Send a crime alert to all connected clients

state (string): State name
crime_type (string): Type of crime
severity (string): 'high', 'medium', or 'low'
socket.emit('crime_alert', { state: 'MAHARASHTRA', crime_type: 'Theft', severity: 'high' });
LISTEN new_alert

Receives new crime alerts broadcast to all clients

state (string): State name
crime_type (string): Type of crime
severity (string): Alert severity level
timestamp (string): ISO timestamp
user (string): Username who triggered alert
socket.on('new_alert', (alert) => { console.log('New alert:', alert); // Display alert to user });
LISTEN error

Receives error messages from the server

message (string): Error description
socket.on('error', (data) => { console.error('Error:', data.message); });

Complete Example

<!DOCTYPE html> <html> <head> <script src="https://cdn.socket.io/4.5.4/socket.io.min.js"></script> </head> <body> <h1>Real-Time Crime Monitor</h1> <div id="userCount">Users: 0</div> <div id="crimeData"></div> <div id="alerts"></div> <script> const socket = io(); socket.on('connect', () => { console.log('Connected!'); socket.emit('request_live_data'); }); socket.on('user_count', (data) => { document.getElementById('userCount').textContent = 'Users: ' + data.count; }); socket.on('live_data_update', (data) => { let html = '<h2>Crime Data ' + data.year + '</h2>'; data.states.forEach((state, i) => { html += '<p>' + state + ': ' + data.crimes[i] + '</p>'; }); document.getElementById('crimeData').innerHTML = html; }); socket.on('new_alert', (alert) => { const alertDiv = document.createElement('div'); alertDiv.innerHTML = '<strong>' + alert.state + '</strong> - ' + alert.crime_type + ' (' + alert.severity + ')'; document.getElementById('alerts') .prepend(alertDiv); }); // Auto-refresh every 30 seconds setInterval(() => { socket.emit('request_live_data'); }, 30000); </script> </body> </html>

REST API Endpoints

POST /api/v1/predict

Get crime predictions for all states

crime_type (string): Type of crime to analyze
fetch('/api/v1/predict', { method: 'POST', headers: {'Content-Type': 'application/json'}, body: JSON.stringify({crime_type: 'Total'}) }) .then(res => res.json()) .then(data => console.log(data));
GET /api/v1/states

Get list of all states

fetch('/api/v1/states') .then(res => res.json()) .then(data => console.log(data.states));
GET /api/v1/crime-types

Get list of all crime types

fetch('/api/v1/crime-types') .then(res => res.json()) .then(data => console.log(data.crime_types));
GET /api/v1/statistics

Get overall crime statistics

fetch('/api/v1/statistics') .then(res => res.json()) .then(data => console.log(data.statistics));

Rate Limits

The API has the following rate limits:

  • 200 requests per day
  • 50 requests per hour
  • Login: 10 requests per minute
  • Register: 5 requests per minute
